## 下载

### Docker

```
docker pull apachecn0/cg-scratch-zh
docker run -tid -p <port>:80 apachecn0/cg-scratch-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g cg-scratch-zh
cg-scratch-zh <port>
# 访问 http://localhost:{port} 查看文档
```
